<!-- GETTING STARTED -->

## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/TheGoldenDev/asrc-react.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start a Node development server
   ```sh
   npm start
   ```
